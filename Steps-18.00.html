Fixed Acidity | Total Wines Chart:

 

  • To construct a Power BI visual that segments wine by fixed acidity, follow these steps:

 Fixed by Total
FA_Segment = SWITCH(
    TRUE(),
    'Wine-Quality'[Fixed Acidity] >= 3.8 && 'Wine-Quality'[Fixed Acidity] < 6.3, "3.8 - 6.2",
    'Wine-Quality'[Fixed Acidity] >= 6.3 && 'Wine-Quality'[Fixed Acidity] < 8.8, "6.3 - 8.7",
    'Wine-Quality'[Fixed Acidity] >= 8.8 && 'Wine-Quality'[Fixed Acidity] < 11.3, "8.8 - 11.2",
    'Wine-Quality'[Fixed Acidity] >= 11.3 && 'Wine-Quality'[Fixed Acidity] < 13.8, "11.3 - 13.7",
    'Wine-Quality'[Fixed Acidity] >= 13.8 && 'Wine-Quality'[Fixed Acidity] <= 15.9, "13.8 - 15.9",
    BLANK()
)






  1. Choose the Line and Stacked Column Chart from the visualizations pane and place it at the desired location in your report, commonly in the top left corner.
  2. Add 'FA_Segment' to the X-axis to categorize the data by fixed acidity ranges. Then, drag 'Total Wines' to the Column y-axis to represent the quantity of wine for each acidity segment.
  3. To analyze how different chemical properties correlate with acidity levels, add measures like 'Citric', 'Volatile', 'Sulphates', and 'Chlorides' to the Line y-axis.
  4. Adjust the chart's title to accurately describe the visual, such as "Fixed by Total."
  5. Format your chart using a consistent color palette, assigning specific colors to each line measure for visual distinction and aligning with the dashboard's color scheme to maintain uniformity.
  • By following these steps, you will create a visual that clearly illustrates the relationship between the fixed acidity of wines and their various chemical compositions.
